# FAQ

Как распечатать?

1. Скачать себе этот репозиторий как файл .zip и распаковать архив
2. Открыть index.html в браузере
3. Нажать Ctrl+P и выбрать принтер "Сохранить как pdf"
4. Распечатать этот pdf на физическом принтере

Как поменять год?

- Сменить год в файлах index.html и script.js

Как добавить свои выходные дни?

- В файле script.js записать список выходных в переменную HOLIDAYS

Как поменять цвета?

- В файле style.css поменять переменные
  --c-text-main,
  --c-text-muted,
  --c-text-light,
  --c-grid,
  --c-grid-light,
  --c-accent
